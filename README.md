# wash-hub-links

Pagina **link-in-bio** di WASH HUB Catania → servita su **links.wash-hub.it** (GitHub Pages).

- `index.html` — pagina statica self-contained (brand Wash Hub, mobile-first)
- `logo.png` — logo
- `CNAME` — dominio custom GitHub Pages

I link interni (wash-hub.it) propagano automaticamente i parametri di tracking
(`utm_*`, `fbclid`, `gclid`, ...) in arrivo, per attribuire le conversioni.
